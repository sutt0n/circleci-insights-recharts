# Setup

1. `npm i`
2. Configure `.env.sample` and rename to `.env`
  * Make sure you grab your [CircleCI API key](https://app.circleci.com/settings/user/tokens)!
3. `npm run get-deployment-data`
4. `npm run get-build-data`
5. `npm run start`
